## Introduction
Robo - Your AI Assistant - это Android-приложение с голосовым ассистентом, предоставляющее широкий спектр возможностей для голосового управления устройством.

## Project Overview
Приложение разработано с использованием современных Android-практик и включает в себя:
- Голосовое управление с поддержкой русского языка
- Интеграцию с системными функциями (звонки, SMS)
- Работу с веб-браузером через Chrome Custom Tabs
- Анализ изображений с камеры
- Специальные возможности доступности для людей с ограниченными возможностями

## Project Setup
### Prerequisites
- Android Studio Arctic Fox или новее
- JDK 11+
- Android SDK (минимальная версия API 24)
- Gradle 7.0+

### Installation
1. Клонировать репозиторий
2. Открыть проект в Android Studio
3. Синхронизировать Gradle файлы
4. Запустить приложение на устройстве или эмуляторе

### Requirements
- Android 7.0 (API 24) или выше
- Доступ к интернету
- Микрофон
- Камера (для функций анализа изображений)
- Доступ к контактам (для функций звонков и SMS)

## Features

### Голосовое управление
#### Распознавание и обработка команд
- Использование Android Speech Recognition API
- Поддержка русского языка
- Автоматическая остановка записи при тишине (3 секунды)
- Оптимизированная обработка голосовых команд

#### Голосовой ответ
- Использование TextToSpeech для синтеза речи
- Настраиваемая скорость речи
- Визуальная индикация процесса записи и воспроизведения

#### Примеры голосовых команд
- "Позвони [имя контакта]" - совершение звонка
- "Отправь SMS [имя контакта] с текстом [сообщение]" - отправка сообщения
- "Открой [url]" - переход по ссылке
- "Найди [запрос]" - поиск в Google
- "Включи камеру" - запуск камеры для анализа
- "Посмотри что на камере" - анализ объектов через камеру

### Телефония и сообщения
#### Телефонные звонки
- Голосовой набор номера
- Поиск контактов по имени
- Подтверждение перед совершением звонка
- Обработка различных формулировок команд

#### SMS-сообщения
- Отправка SMS на номер или контакту
- Поддержка произвольного текста сообщения
- Проверка и подтверждение перед отправкой
- Защита от случайной отправки

### Веб-браузер
#### Интеграция с Chrome Custom Tabs
- Встроенный просмотр веб-страниц
- Сохранение контекста приложения
- Быстрая загрузка страниц

#### Обработка URL и поисковых запросов
- Поддержка прямых URL ("открой google.com")
- Поддержка поисковых запросов ("найди рецепт борща")
- Автоматическое добавление https:// к ссылкам
- Fallback на браузер по умолчанию, если Chrome не установлен

### Работа с камерой
#### Анализ изображений
- Интеграция с CameraX для работы с камерой
- Использование ML-моделей для распознавания объектов
- Голосовое описание распознанных объектов

#### Специальные возможности камеры
- Автоматическая съемка при обнаружении объекта
- Голосовые подсказки при работе с камерой
- Вибрационная обратная связь

### Доступность
#### Настройки доступности
- Изменение размера текста (очень маленький, маленький, нормальный, большой, очень большой)
- Высококонтрастный режим
- Голосовая обратная связь
- Вибрационная обратная связь

#### Система тем
- Обычная тема
- Высококонтрастная тема для людей с нарушениями зрения
- Переключение через настройки или меню
- Сохранение выбранной темы

#### Адаптация интерфейса
- Специальная структура для screen readers
- Описательные метки для элементов управления
- Логичная навигация с клавиатуры

## Technical Details

### Architecture
Приложение использует MVVM-архитектуру (Model-View-ViewModel):
- **Model**: Классы данных в пакете `model`
- **View**: XML-разметка и Activity классы
- **ViewModel**: Классы в пакете `viewmodel` для обработки бизнес-логики

### Используемые технологии и библиотеки
- **Язык программирования**: Java
- **Jetpack компоненты**:
  - ViewModel
  - LiveData
  - ViewBinding
  - Preferences
- **Сеть**:
  - Retrofit для API-запросов
  - OkHttp для HTTP-клиента
  - Gson для сериализации/десериализации JSON
- **UI**:
  - Material Design компоненты
  - RecyclerView для отображения списков
  - CardView для карточек сообщений
  - ConstraintLayout для гибкой верстки
- **Мультимедиа**:
  - CameraX для работы с камерой
  - TextToSpeech для синтеза речи
  - SpeechRecognizer для распознавания речи
- **Изображения**:
  - Glide для загрузки и кэширования изображений
- **Веб**:
  - Chrome Custom Tabs для встроенного веб-браузера
- **Хранение данных**:
  - SharedPreferences для настроек
  - Сериализация/десериализация для хранения истории чатов

### Интеграция с API
Приложение интегрируется с OpenRouter API для обработки чат-запросов:
- Использование Retrofit для вызовов API
- Отправка и получение сообщений через RESTful API
- Обработка ошибок и повторные попытки
- Поддержка мультимодальных сообщений (текст + изображения)

### Система разрешений
Приложение запрашивает следующие разрешения:
- `RECORD_AUDIO`: для распознавания речи
- `READ_CONTACTS`: для доступа к контактам
- `CALL_PHONE`: для совершения звонков
- `SEND_SMS`: для отправки сообщений
- `READ_PHONE_STATE`: для работы с телефонией
- `CAMERA`: для доступа к камере
- `INTERNET`: для работы с сетью
- `VIBRATE`: для тактильной обратной связи

## Build and Run
### Сборка через Android Studio
1. Откройте проект в Android Studio
2. Выберите Build > Make Project для сборки проекта
3. Выберите Run > Run 'app' для запуска на устройстве или эмуляторе

### Сборка через командную строку
```bash
# Сборка отладочной версии
./gradlew assembleDebug

# Сборка релизной версии
./gradlew assembleRelease

# Установка на подключенное устройство
./gradlew installDebug
```

### Конфигурация релизных сборок
Для создания релизной версии требуется:
1. Создать keystore файл для подписи приложения
2. Указать параметры подписи в файле `gradle.properties` или через переменные окружения
3. Запустить сборку через `./gradlew assembleRelease`
